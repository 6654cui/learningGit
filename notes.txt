初始化仓库：
    git init

提交代码：
    git add 修改的文件
    git commit -m "备注"

检查仓库状态：
    git status

检查文件修改：
    git diff 文件名

查看提交日志：
    git log
    git log --pretty=oneline(显示成一行)

版本回退：
    git reset --hard 版本号

查看历史命令：
    git reflog

查看工作区和版本库的区别：
    git diff HEAD -- 文件名

撤销工作区修改：
    git checkout -- 文件名 （撤销到最后一次add/commit的状态）

撤销暂存区修改：
    git reset HEAD 文件名
